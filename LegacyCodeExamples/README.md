# Legacy Code Examples

This directory contains Python code examples for each real-life scenario mentioned in Chapter 4: Legacy Code.

Each file corresponds to a section:

- `01_WhatIsLegacyCode.py`: Banking transaction and e-commerce payment examples.
- `02_TheFirstStepInRefactoring.py`: Web app authentication and data pipeline examples.
- `03_Seams.py`: Logging and database access seams.
- `04_BasicStrategy.py`: CRM email and file parser strategy.
- `05_ExtractAndOverrideCall.py`: Web scraper and report generator examples.
- `06_CodeCoverage.py`: Input validation and interest calculation for coverage.
- `07_CharacterizationTests.py`: Button rendering and data export snapshots.
- `08_FixingBugsTestFirst.py`: User profile loader and tax calculation bug fixes.
- `09_AddingNewFeaturesTestFirst.py`: OAuth login and product search filter additions.

Run with Python 3. Some examples include commented test code; uncomment and use pytest to run tests.
